Bill Pro License Generator

Generate a signed Bill Pro activation code for a specific device ID. Keep the private JWK outside your public app files.

Steps

1. Open Bill Pro in the app and copy the device ID.

2. Load bill-pro-license-private.jwk into this page.

3. Enter the device ID and optional shop name, then generate the activation code.

4. Paste the activation code back into Bill Pro and activate it.

The signed payload includes edition, deviceId, shopName, and issuedAt.
